Multiverso Rick e Morty

Aplicação web desenvolvida com React para consumo da API pública de Rick and Morty.

![Image](https://github.com/user-attachments/assets/a2ccb912-6ca5-44d8-823f-3a7f8b9e6331)


Tecnologias utilizadas:

React, React Router DOM, API pública de Rick and Morty

API utilizada:

Os dados são consumidos da API pública:

https://rickandmortyapi.com/

A API fornece informações como: Nome do personagem, Status (vivo, morto ou desconhecido), Espécie, Gênero, Origem, Imagem e etc
